Importar plantilla de AFFiNE.pro a tu instancia

Esta página reescribe el link de una plantilla de affine.pro para que se importe en tu Affine self-hosted, usando tu proxy CDN con CORS habilitado.

Cómo usar:

  1. Andá a affine.pro/templates
  2. Abrí cualquier plantilla. Right-click sobre el botón "Use this template" → "Copy link"
  3. Pegá el link acá abajo → click "Importar"
